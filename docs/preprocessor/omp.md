# <a name="omp-no-locpragma"></a>`omp` pragma

省略可能なディレクティブの句と共に、1 つ以上の OpenMP ディレクティブを受け取ります。

## <a name="syntax"></a>構文

> **`#pragma omp`***ディレクティブ*
